
# NEPSE WACC Calculator

A simple web app to visualize the Weighted Average Cost per Share (WACC) for NEPSE stock purchases. Enter your share price, quantity, and step to see the cost breakdown and chart.

## Features
- Input share price, quantity, and step size
- See cost breakdown (trade value, brokerage, SEBON fee, DP fee)
- Visualize cost per share for different quantities
- Responsive, modern UI

## Getting Started

### Prerequisites
- Node.js (v16 or newer recommended)

### Installation
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd nepse-wacc-graph
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production
```bash
npm run build
```
